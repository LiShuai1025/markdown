flowchart TD
    A[开始：准备外出用餐] --> B{是否已有目标餐厅？}

    B -- 否 --> C[根据菜系 / 距离 / 预算筛选餐厅]
    C --> D[查看餐厅评价与环境]
    D --> E[选择一家餐厅]

    B -- 是 --> E[进入已选餐厅]

    E --> F{是否需要安静或私密空间？}
    
    F -- 是 --> G{同行人数 ≥ 4？}
    G -- 是 --> H[优先考虑订包间]
    G -- 否 --> I{是否有商务/重要聚会？}
    I -- 是 --> H
    I -- 否 --> J[大厅安静角落即可]

    F -- 否 --> K{是否高峰时段？}
    K -- 是 --> L{包间是否需要最低消费？}
    L -- 能接受 --> H
    L -- 不能接受 --> M[选择大厅]
    K -- 否 --> M

    H --> N[提前电话 / App 订包间]
    M --> O[到店后大厅排队或直接入座]

    N --> P[用餐]
    O --> P
    P --> Q[结束]
